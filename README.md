# morandi.nvim

A soft, muted Neovim colorscheme inspired by Morandi tones.

## Installation

### lazy.nvim

```lua
{ "morandi-theme/morandi.nvim" }
```

### packer.nvim

```lua
use "morandi-theme/morandi.nvim"
```

## Usage

```vim
colorscheme morandi
```

You can set it in your `init.lua`:

```lua
vim.cmd.colorscheme("morandi")
```
